# maiagent-project

## 設定

1. 複製 `.env.example` 為 `.env`：
   ```
   cp .env.example .env
   ```

2. 編輯 `.env` 檔案，填入您的 API 金鑰和其他敏感資訊

3. 從 GitHub 下載並安裝：
   ```
   git clone https://github.com/microsoft/markitdown.git
   cd markitdown
   ```

4. 安裝依賴套件：
   ```
   pip install -r requirements.txt
   ```
